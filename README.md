# Word-Counter
🔤 WordCounter
A beginner-friendly Java Word Counter application with a simple Swing-based GUI that allows users to enter text and instantly calculate the total number of words.
✨ Features
🖥️ Simple Java Swing GUI
✍️ Text input area
🔘 Button-based word counting
📊 Instant word count display
⚡ Beginner-friendly logic
📚 Uses BufferedReader and String processing
🛠️ Technologies Used
Java
Java Swing
BufferedReader
Event Handling
String Manipulation
🚀 How It Works
User enters text in the text area
Clicks the Count Words button
The application processes the text
Total word count is displayed instantly
▶️ Run the Application
Install Java JDK
Save the file as WordCounter.java
Open terminal or command prompt
Compile the program:
Bash
javac WordCounter.java
Run the program:
Bash
java WordCounter
